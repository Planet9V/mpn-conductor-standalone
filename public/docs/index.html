MPN Conductor

Musical Psychometric Notation System - Standalone Docker Application

Overview

MPN Conductor is an advanced visualization and audio synthesis system that translates psychological states into musical notation using the McKenney-Lacan theoretical framework. This standalone application provides a complete multi-stave conductor's score with real-time psychometric analysis.

Current Version

v3.0 | Canon v2.4 | 132 Passing Tests | Docker Ready

Technology Stack

  • Framework: Next.js 16.1 with React 19
  • Language: TypeScript 5
  • Styling: Tailwind CSS 4
  • Audio: Tone.js
  • 3D Graphics: react-three-fiber, Three.js
  • Visualization: Nivo, XYFlow, react-globe.gl
  • Testing: Vitest, Playwright
  • Deployment: Docker, PostgreSQL
