McKenney-Lacan Theory

Theoretical foundations of Musical Psychometric Notation

Overview

The McKenney-Lacan theory provides a formal framework for translating psychological states into musical notation. It combines:

  • Riemannian Geometry - Trauma as manifold curvature
  • Information Theory - Entropy as rhythmic disorder
  • Lacanian Psychoanalysis - RSI registers as modal selection
  • Neo-Riemannian Theory - Harmonic transformations
  • Dynamical Systems - Lyapunov stability indicators

Core Metrics

Trauma (τ) - Riemann Curvature

Trauma measures the "curvature" of psychological space:

τ = Σ(keyword_weights) / max_possible

τ ∈ [0, 1]
τ = 0: Flat manifold (no trauma)
τ = 1: Maximum curvature (crisis)

Musical Mapping: τ → Dynamics (pp to ff)

Entropy (H) - Shannon Entropy

Entropy measures disorder in the narrative:

H = -Σ p(x) log₂ p(x)

H → normalized to [0, 1]
H = 0: Complete order
H = 1: Maximum disorder

Musical Mapping: H → Tempo and Time Signature

RSI Registers - Lacanian Topology

The Real, Symbolic, and Imaginary registers from Lacan:

Register Keywords Mode
Real (R) death, trauma, blood, void Phrygian, Locrian
Symbolic (S) law, order, king, father Ionian, Mixolydian
Imaginary (I) mirror, image, self, beauty Lydian, Dorian

Lyapunov Exponent (λ)

Measures system stability based on trauma and entropy:

λ = (τ + H - 0.5) × 0.5

λ < 0:   STABLE   (convergent)
λ ≈ 0:   EDGE     (critical point)
λ > 0:   CHAOTIC  (divergent)

Musical Mapping: λ → Harmonic tension, chord complexity

Neo-Riemannian Operations

Chord transformations based on psychological state:

Operation Description Trigger
P (Parallel) Major ↔ minor Real dominant
L (Leading-tone) C → e, e → C Imaginary dominant
R (Relative) C → a, a → C Symbolic dominant
PLP Compound operation High trauma plateau

DISC → Instrument Mapping

Personality profiles map to instrumental families:

DISC Type Characteristics Instruments
D (Dominance) Direct, decisive, competitive Brass (trumpet, trombone)
I (Influence) Enthusiastic, optimistic, social Woodwinds (flute, clarinet)
S (Steadiness) Patient, reliable, supportive Strings (cello, viola)
C (Conscientiousness) Analytical, systematic, precise Piano, harpsichord

Borromean Stability Index (BSI)

Topological measure of register balance:

BSI = max(0.3, 1 - |λ|)

BSI → [0.3, 1.0]
BSI = 1.0: Perfect Borromean link
BSI = 0.3: Minimum stability
