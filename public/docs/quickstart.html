Quick Start Guide

Get MPN Conductor running in under 5 minutes

Prerequisites

  • Node.js 20.0.0 or higher
  • npm 9.0+ or equivalent package manager
  • Git for cloning the repository
  • Docker (optional, for containerized deployment)
Recommended

Use Node.js 20 LTS for best compatibility with Next.js 16 and React 19.

Installation

1. Clone the Repository

git clone https://github.com/your-org/mpn-conductor-standalone.git
cd mpn-conductor-standalone

2. Install Dependencies

npm install
Peer Dependency Note

If you encounter peer dependency warnings, the project includes an .npmrc file with legacy-peer-deps=true to handle this automatically.

3. Configure Environment Variables

# Copy the example environment file
cp .env.example .env.local

# Edit with your API keys (optional for basic usage)
nano .env.local
Variable Required Description
OPENROUTER_API_KEY Optional For AI-enhanced melody generation
HUGGINGFACE_API_KEY Optional For music model inference
ELEVENLABS_API_KEY Optional For text-to-speech narration
DATABASE_URL Optional PostgreSQL connection string

Running the Application

Development Mode

# Start development server
npm run dev

# Or on a specific port
npm run dev -- --port 3002

Open http://localhost:3000 in your browser.

Production Build

# Build for production
npm run build

# Start production server
npm start

Docker Deployment

# Build and run with Docker Compose
docker-compose up -d

# View logs
docker-compose logs -f mpn-app

Application will be available at http://localhost:3001

Docker Services

Docker Compose starts both the Next.js application and a PostgreSQL database with the MPN schema pre-configured.

First Steps

1. Landing Page

Navigate to / to see the MPN landing page with links to all sections.

2. Conductor Score

Go to /mpn-conductor to access the main conductor's score interface:

  • Select a literary scenario (Hamlet, Macbeth, Oedipus)
  • Press Play to auto-advance through frames
  • Enable Audio for real-time synthesis
  • Toggle AI for enhanced melody generation

3. Reference Dictionary

Visit /mpn-reference to explore all musical-psychometric mappings.

4. Lab Experiments

Check /mpn-lab for experimental visualizations and tools.

Running Tests

# Run unit tests
npm run test

# Run with UI
npm run test -- --ui

# Run E2E tests
npm run test:e2e
