## A

**Accelerando**  
Gradual increase in tempo. See [21_REF_RHYTHM_DETAILED](./21_REF_RHYTHM_DETAILED.md).

**ADSR**  
Attack-Decay-Sustain-Release envelope describing timbre shape. 
See [20_REF_TIMBRE_DETAILED](./20_REF_TIMBRE_DETAILED.md).

**Articulation**  
Manner of note attack and release (legato, staccato, marcato). 
See [28_REF_ARTICULATION_DETAILED](./28_REF_ARTICULATION_DETAILED.md).

---

## B

**Big Five** (OCEAN)  
Five-factor personality model: Openness, Conscientiousness, Extraversion, 
Agreeableness, Neuroticism. See [03_PSYCHOMETRIC_THEORY](./03_PSYCHOMETRIC_THEORY.md#big-five-ocean).

**Borromean Knot**  
Three interlocked rings representing RSI registers. 
See [02_BORROMEAN_TOPOLOGY](./02_BORROMEAN_TOPOLOGY.md).

**BSI** (Borromean Stability Index)  
Measure of RSI register balance: $\text{BSI} = \min(r,s,i) / \max(r,s,i)$. 
See [02_BORROMEAN_TOPOLOGY](./02_BORROMEAN_TOPOLOGY.md#stability-measure).

---

## C

**Cents**  
Logarithmic unit of pitch interval: 100 cents = 1 semitone. 
Formula: $c = 1200 \log_2(f_2 / f_1)$.

**Chromatic**  
Using all 12 semitones. See [26_REF_MODE_DETAILED](./26_REF_MODE_DETAILED.md).

**Consonance**  
Perceptually stable intervals (octave, P5, M3). 
See [27_REF_INTERVALS_DETAILED](./27_REF_INTERVALS_DETAILED.md).

---

## D

**Dark Triad**  
Three subclinical personality traits: Narcissism, Machiavellianism, 
Psychopathy. See [03_PSYCHOMETRIC_THEORY](./03_PSYCHOMETRIC_THEORY.md#dark-triad).

**DISC**  
Behavioral assessment: Dominance, Influence, Steadiness, Compliance. 
See [03_PSYCHOMETRIC_THEORY](./03_PSYCHOMETRIC_THEORY.md#disc-theory).

**Dissonance**  
Perceptually unstable intervals (m2, tritone). 
See [27_REF_INTERVALS_DETAILED](./27_REF_INTERVALS_DETAILED.md).

**Dynamics**  
Volume/intensity markings (pp ‚Üí fff). 
See [23_REF_DYNAMICS_DETAILED](./23_REF_DYNAMICS_DETAILED.md).

---

## E

**Entropy ($H$)**  
Shannon entropy measuring unpredictability: $H = -\sum p_i \log_2 p_i$. 
See [11_ENTROPY_CALCULUS](./11_ENTROPY_CALCULUS.md).

---

## F

**Fermata**  
Held note of indeterminate duration. 
See [29_REF_SILENCE_DETAILED](./29_REF_SILENCE_DETAILED.md).

---

## G

**General Pause (G.P.)**  
Complete silence by all instruments. 
See [29_REF_SILENCE_DETAILED](./29_REF_SILENCE_DETAILED.md).

---

## H

**Homophonic**  
Texture with melody and accompaniment. 
See [25_REF_TEXTURE_DETAILED](./25_REF_TEXTURE_DETAILED.md).

## I

**Imaginary (ùìò)**  
Lacanian register of ego, image, mirror stage. 
See [01_LACANIAN_FRAMEWORK](./01_LACANIAN_FRAMEWORK.md#the-imaginary).

**Ionian**  
Major scale mode. See [26_REF_MODE_DETAILED](./26_REF_MODE_DETAILED.md).

## L

**L Transform**  
Leading-tone Neo-Riemannian operation. 
See [13_NEO_RIEMANNIAN_MATH](./13_NEO_RIEMANNIAN_MATH.md#leading-tone-exchange-l).

**Leitmotif**  
Recurring musical theme associated with character/idea. 
See [05_LEITMOTIF_THEORY](./05_LEITMOTIF_THEORY.md).

**Locrian**  
Darkest diatonic mode (diminished 5th). 
See [26_REF_MODE_DETAILED](./26_REF_MODE_DETAILED.md).

**Lydian**  
Bright mode with raised 4th. See [26_REF_MODE_DETAILED](./26_REF_MODE_DETAILED.md).

**Lyapunov Exponent (Œª)**  
Chaos measure: $\lambda = (\tau + H - 0.5) / 2$. 
See [12_LYAPUNOV_STABILITY](./12_LYAPUNOV_STABILITY.md).

## P

**P Transform**  
Parallel Neo-Riemannian operation (major ‚Üî minor). 
See [13_NEO_RIEMANNIAN_MATH](./13_NEO_RIEMANNIAN_MATH.md#parallel-p).

**Phrygian**

## S

**Seldon Crisis**  
Critical system state: BSI < 0.3 AND Œª > 0.2. 
See [12_LYAPUNOV_STABILITY](./12_LYAPUNOV_STABILITY.md#crisis-detection).

**Staccato**  
Short, detached articulation. 
See [28_REF_ARTICULATION_DETAILED](./28_REF_ARTICULATION_DETAILED.md).

**Symbolic (ùì¢)**

## T

**Tempo**  
Speed in beats per minute (BPM). 
See [21_REF_RHYTHM_DETAILED](./21_REF_RHYTHM_DETAILED.md).

**Timbre**  
Tone color/quality of sound.

**Tritone**

See [27_REF_INTERVALS_DETAILED](./27_REF_INTERVALS_DETAILED.md).
