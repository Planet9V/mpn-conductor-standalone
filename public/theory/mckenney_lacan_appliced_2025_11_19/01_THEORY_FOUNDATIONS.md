# 01. Theory Foundations
## McKenney-Lacan Applied Theory: Theoretical Basis

---

## 1. Introduction

The McKenney-Lacan Applied Theory represents a novel integration of:
- **Lacanian psychoanalysis** (Symbolic, Imaginary, Real registers)
- **Psychometric profiling** (DISC, Big Five, Dark Triad)
- **Musical theory** (harmony, rhythm, dynamics, orchestration)
- **Information theory** (entropy, signal processing)

The core premise is that human psychological states can be systematically 
translated into musical parameters, creating an objective "score" of human 
interaction and emotional dynamics.

---

## 2. Foundational Theories

### 2.1 Lacanian Registers (RSI)

Jacques Lacan proposed three registers of human experience:

| Register | Description | Musical Expression |
|----------|-------------|-------------------|
| **Real** | The unsymbolizable, trauma | Dark modes (Phrygian, Locrian) |
| **Symbolic** | Language, law, order | Major/Ionian mode |
| **Imaginary** | Ego, fantasy, image | Lydian, whole-tone scales |

**Reference**: Lacan, J. (1977). *Écrits: A Selection*. Norton.

### 2.2 Borromean Knot Stability

The three registers interlock like Borromean rings. Psychological stability 
requires all three. The system measures this via:

```
Stability = min(Real, Symbolic, Imaginary) / max(Real, Symbolic, Imaginary)
```

When one register dominates excessively, instability results.

**Reference**: RSCH-01-BORROMEAN_STABILITY.md

### 2.3 Object petit a (objet a)

The unattainable object of desire that drives human behavior. Mapped to:
- **Dominant 7th chords** (wanting resolution)
- **Unresolved cadences**
- **Ascending melodies** (reaching for something)

**Reference**: RSCH-10-OBJECT_PETIT_A.md

## 3. Psychometric Frameworks

### 3.1 DISC Theory

William Marston's DISC model profiles behavioral tendencies:

| Factor | Description | Musical Expression |
|--------|-------------|-------------------|
| **D** (Dominance) | Assertive, decisive | Brass, staccato |
| **I** (Influence) | Expressive, enthusiastic | Woodwinds, legato |
| **S** (Steadiness) | Patient, supportive | Strings, sustained |
| **C** (Compliance) | Analytical, careful | Keyboard, precise |

**Reference**: Marston, W.M. (1928). *Emotions of Normal People*.

### 3.2 Big Five (OCEAN)

The Five Factor Model of personality:

| Factor | High Expression | Low Expression |
|--------|-----------------|----------------|
| **Openness** | Complex harmonies | Simple triads |
| **Conscientiousness** | Regular rhythm | Syncopation |
| **Extraversion** | Loud dynamics | Soft dynamics |
| **Agreeableness** | Legato, consonance | Staccato, dissonance |
| **Neuroticism** | Unstable tempo | Stable tempo |

**Reference**: Costa, P.T., & McCrae, R.R. (1992). NEO-PI-R Professional Manual.

### 3.3 Dark Triad

Subclinical personality traits with distinct musical signatures:

| Trait | Musical Expression |
|-------|-------------------|
| **Narcissism** | Loud, ignores ensemble, sharp tuning (+5 cents) |
| **Machiavellianism** | Muted, detuned (-10 cents), hidden motifs |
| **Psychopathy** | No vibrato, mechanical precision, cold timbre |

**Reference**: RSCH-33-DARK_TRIAD.md

## 4. Integration: The Psychometric Calculus

The system converts psychometric states to music through a calculus:

```typescript
Musical_Output = f(Trauma, Entropy, RSI, DISC, OCEAN, DarkTriad)
```

Key transformations:
- **Trauma → Dynamics**: Higher trauma = louder volume
- **Entropy → Rhythm**: Higher entropy = more irregular time
- **RSI → Mode**: Register dominance selects scale
- **DISC → Instrument**: Profile selects instrument family

**Reference**: RSCH-43-PSYCHOMETRIC_CALCULUS.md

## 5. The Seven-Staff Fugue Model

Each dramatic scene is orchestrated as a seven-staff score:

| Staff | Content |
|-------|---------|
| 1-6 | Individual actors (up to 6) |
| 7 | Environmental/atmospheric layer |

Each actor has:
- **Leitmotif**: Personal musical theme
- **Transformation state**: Original, inverted, fragmented
- **Instrument**: Based on DISC profile
- **Dynamic level**: Based on trauma

**Reference**: RSCH-41-SEVEN_STAFF_FUGUE.md
