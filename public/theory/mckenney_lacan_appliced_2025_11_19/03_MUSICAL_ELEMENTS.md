# 03. Musical Elements
## Categories of Musical Expression in MPN

---

## Overview

The MPN system organizes musical elements into 10 categories.
Each category maps to specific psychometric dimensions.

---

## Category Summary

| ID | Category | Elements | Primary Dimension |
|----|----------|----------|-------------------|
| 1 | Timbre | Instruments, tone color | DISC |
| 2 | Rhythm | Time, tempo, meter | Entropy |
| 3 | Harmony | Chords, progressions | Relationship |
| 4 | Dynamics | Volume, intensity | Trauma |
| 5 | Melody | Leitmotif, contour | Identity |
| 6 | Texture | Orchestration density | Relationship |
| 7 | Mode | Scales, modes | Lacanian |
| 8 | Intervals | Note distances | Relationship |
| 9 | Articulation | Attack, phrasing | DISC/OCEAN |
| 10 | Silence | Rests, pauses | Stability |

---

## 1. Timbre (Tone Color)

### Definition
The characteristic quality of a musical sound.

### Subcategories

| Subcategory | Description |
|-------------|-------------|
| instrument_family | Brass, woodwind, string, keyboard |
| timbre_modulation | Mutes, detuning, effects |
| register | High/low frequency range |
| adsr_envelope | Attack-Decay-Sustain-Release shape |

### Key Mappings

| DISC Type | Instrument | Rationale |
|-----------|------------|-----------|
| D (Dominant) | Brass | Bold, commanding sound |
| I (Influential) | Woodwind | Expressive, singing quality |
| S (Steady) | Strings | Warm, sustained support |
| C (Compliant) | Keyboard | Precise, analytical control |

---

## 2. Rhythm (Time Organization)

### Definition
The pattern of sounds and silences in time.

### Subcategories

| Subcategory | Description |
|-------------|-------------|
| time_signature | Meter (4/4, 3/4, 5/4, etc.) |
| tempo | Speed in BPM |
| syncopation | Off-beat accents |
| polyrhythm | Multiple simultaneous patterns |
| tempo_change | Accelerando, ritardando |

### Entropy Mapping

| Entropy | Time Signature | Character |
|---------|---------------|-----------|
| Low (< 0.3) | 4/4 | Stable, predictable |
| Medium (0.3-0.6) | 3/4, 6/8 | Flowing, dance-like |
| High (0.6-0.85) | 5/4, 7/8, 11/8 | Unsettled, anxious |
| Critical (> 0.85) | Free meter | Chaotic, unpredictable |

---

## 3. Harmony (Vertical Structure)

### Definition
Simultaneous combination of notes forming chords.

### Subcategories

| Subcategory | Description |
|-------------|-------------|
| chord_type | Major, minor, diminished, augmented |
| chord_progression | Sequence of chords |
| voicing | Distribution of notes |
| neo_riemannian | Transformational operations |

### Tension Mapping

| Chord Type | Tension Level | Usage |
|------------|--------------|-------|
| Major/Major7 | Low | Resolution, peace |
| Minor/Minor7 | Medium-Low | Sadness, depth |
| Dominant7 | Medium | Anticipation |
| Diminished | High | Anxiety, threat |
| Augmented/Tritone | Maximum | Crisis, rupture |

---

## 4. Dynamics (Volume)

### Definition
The loudness or softness of musical sound.

### Subcategories

| Subcategory | Description |
|-------------|-------------|
| volume_level | Static dynamic marking |
| dynamic_change | Crescendo, decrescendo |
| accent | Sforzando, marcato |

### Trauma Mapping

| Trauma | Dynamic | Velocity | Label |
|--------|---------|----------|-------|
| 0.0-0.2 | ppp/pp | 20-45 | Very soft |
| 0.2-0.4 | p/mp | 45-65 | Soft |
| 0.4-0.6 | mf | 65-85 | Medium |
| 0.6-0.8 | f/ff | 85-110 | Loud |
| 0.8-1.0 | fff | 110-127 | Very loud |

---

## 5. Melody (Horizontal Structure)

### Definition
A sequence of single pitches forming a musical line.

### Subcategories

| Subcategory | Description |
|-------------|-------------|
| contour | Shape: ascending, descending, static |
| leitmotif_transformation | Original, inverted, fragmented |
| interval_profile | Jump sizes between notes |

### Leitmotif States

| State | Trauma | Entropy | Description |
|-------|--------|---------|-------------|
| Original | < 0.3 | < 0.4 | Normal, stable |
| Inverted | > 0.6 | any | Mirror intervals |
| Retrograde | any | > 0.5 | Played backwards |
| Fragmented | > 0.5 | > 0.6 | Notes missing |
| Augmented | any | < 0.3 | Stretched time |
| Diminished | > 0.4 | > 0.5 | Compressed time |

---

## 6. Texture (Density)

### Definition
The way melodic, rhythmic, and harmonic elements combine.

### Subcategories

| Subcategory | Description |
|-------------|-------------|
| orchestral_texture | Mono-, homo-, polyphonic |
| background_type | Drone, ostinato, silence |
| density | Number of simultaneous voices |

### Relationship Mapping

| Texture | Actors | Relationship |
|---------|--------|--------------|
| Monophonic | 1 | Solo, isolation |
| Homophonic | 2+ | Consensus, alignment |
| Polyphonic | 2+ | Debate, competition |
| Heterophonic | 2+ | Variation, individuality |

---

## 7. Mode (Scale)

### Definition
The collection of pitches used as the basis for melody/harmony.

### Subcategories

| Subcategory | Description |
|-------------|-------------|
| scale_mode | Ionian, Dorian, Phrygian, etc. |
| chromaticism | Use of all 12 notes |

### Lacanian Mapping

| Register | Modes | Character |
|----------|-------|-----------|
| Symbolic | Ionian, Lydian | Bright, ordered |
| Imaginary | Lydian, Whole-tone | Dreamy, ambiguous |
| Real | Phrygian, Locrian | Dark, traumatic |

---

## 8. Intervals (Distance)

### Definition
The distance between two pitches.

### Subcategories

| Subcategory | Description |
|-------------|-------------|
| consonance | Stable intervals |
| dissonance | Unstable intervals |

### Relationship Mapping

| Interval | Cents | Relationship |
|----------|-------|--------------|
| Unison/Octave | 0/1200 | Identity |
| Perfect 5th | 700 | Alliance |
| Major 3rd | 400 | Cooperation |
| Minor 2nd | 100 | Tension |
| Tritone | 600 | Conflict |

---

## 9. Articulation (Attack)

### Definition
The manner in which notes are performed.

### Subcategories

| Subcategory | Description |
|-------------|-------------|
| attack_style | Staccato, legato, marcato |
| phrasing | Musical sentence structure |

### DISC Mapping

| DISC | Articulation | Character |
|------|--------------|-----------|
| D | Staccato | Sharp, decisive |
| I | Legato | Smooth, connected |
| S | Tenuto | Sustained, patient |
| C | Precise | Measured, exact |

---

## 10. Silence (Rest)

### Definition
The absence of sound as a musical element.

### Subcategories

| Subcategory | Description |
|-------------|-------------|
| rest_type | Duration of silence |
| fermata | Extended pause |
| general_pause | All instruments stop |

### Psychological Mapping

| Silence Type | Duration | Meaning |
|--------------|----------|---------|
| Short rest | 1-2 beats | Natural breath |
| Long rest | 4+ beats | Contemplation |
| Fermata | Unmeasured | Suspended time |
| General Pause | All stop | Shock, rupture |
